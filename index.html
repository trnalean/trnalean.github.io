百度云不限速下载教程

百度云不限速下载教程

  • 写在开始的话：

准备工具:

  1. aria2
  2. Microsoft Edge浏览器（谷歌浏览器）

实现原理：

浏览器安装油猴插件，安装已经集成好的百度云直链提取插件，通过aria2（推荐）或IDM等多线程下载器通过直链下载达到满速。
< !–more–>

此方法单日内下载流量过大有封号可能，开通一次会员即可解除封号且无限制提取直链

解释：提取直链为个人百度云账号内提取，单账号短时间大流量提取直链可能被限速或封号。
本人亲测单日100G内绝无问题。

教程正式开始：

1.安装Tampermonkey（油猴）插件

方法A：Microsoft Edge浏览器点击插件管理。

点击 获取 Microsoft Edge 扩展
搜索 Tampermonkey，点击安装即可。

方法B：浏览器搜索Tampermonkey，进入官网自行安装。
此时你已经安装好油猴插件管理，现在去下载百度云直链提取插件。

2.安装百度云直链提取插件

浏览器搜索 greasy fork 或 点击我
进入greasy fork官网
搜索 百度云下载——————有多个脚本可以使用

这里推荐使用此脚本 点我下载

ps：如果不安装好Tampermonkey，此时点击下载浏览器会自动下载js文件，无法使用，如正确安装Tampermonkey，点击安装会自动进入脚本安装管理界面。

3.安装aria2（多线程下载程序）

ps：此程序请务必安装教程配置，否则无法正常使用。

使用须知：aria2为下载器本体，ariaNG为下载器web本地管理页面

  1. 下载本人打包好的aria2（已经集成ariaNG管理页面）解压码:linpoodown

    下载软件 进入QQ群获取:https://jq.qq.com/?_wv=1027&k=5rwWM8Bp
    1. 打开软件本体aria2.exe（非arai2c.exe），此时右下角任务栏出现软件图标即开启成功
    2. 点击 点我启动.html，进入网页web管理页面，可使用Microsoft Edge浏览器或任意浏览器打开即可
    3. web管理页面显示Aria2已连接，即可正常使用。

  2. 进入aria2NG管理页面，Aria2设置，点击基本设置，设置下载路径

    ps：必须按自己电脑重新指定下载路径，否则无法开始下载。

  3. 进入他人分享的百度云分享链接页面，选中你想下载的文件，登录自己百度云账号后保存至自己账号中

  4. 选中想下载的文件

    1. 点击简易下载助手
    2. 点击获取直链地址
    3. 点击发送至Aria2

  5. 此时，切换至aria2NG网页控制面板，即可看见正在文件，效果如图

结语

本方法能高速下载大文件，游戏，电影等，但两日内尽量不要下载超过100G（参考阿里网盘风控）
